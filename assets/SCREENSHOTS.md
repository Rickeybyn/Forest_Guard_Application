# ForestGuard Screenshots Guide

This directory should contain screenshots and UI previews for the ForestGuard application.

## Screenshots to Add

### Authentication Screens
- **login_screen.png** — User login interface
- **signup_screen.png** — User registration interface

### Citizen Flow Screens
- **dashboard_citizen.png** — Main citizen dashboard
- **select_incident_type.png** — Incident category selection
- **photo_capture_screen.png** — Photo capture interface
- **location_screen.png** — GPS location display
- **details_form.png** — Incident details input form
- **review_summary.png** — Confirmation review screen
- **confirmation_screen.png** — Success confirmation with tracking ID
- **my_reports_timeline.png** — Status tracking view

### Officer Dashboard
- **officer_dashboard.png** — Forest officer incident management interface
- **incident_details_view.png** — Detailed incident information

### Additional Screens
- **education_tips.png** — Educational content section
- **app_navigation_flow.png** — App navigation structure diagram

## Adding Screenshots

1. **Capture screenshots** from running the app in Android Studio emulator or device
2. **Save as PNG** with appropriate names matching above list
3. **Place in this directory** (assets/screenshots/)
4. **Update README.md** with image references using relative paths
5. **Commit and push** changes to GitHub

## Naming Convention
Use lowercase with underscores: `feature_name_screen.png`

## Recommended Dimensions
- **Portrait orientation** (9:16 aspect ratio)
- **Resolution**: 1080 x 1920 px (full HD)
- **File size**: Keep under 500 KB per image
